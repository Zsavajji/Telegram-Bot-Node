Telegram-Bot-Node

A plugin-based, all-in-one Telegram bot written in Node.js.


What is Telegram-Bot-Node?

Telegram-Bot-Node is a Telegram bot framework written in Node.js. Its architecture is based on plugins, many of which are bundled by default. It supports:

  • All bot API features
  • Moderation tools
  • Enabling, disabling and live-reloading plugins
  • Importing custom plugins
  • Automatic software updates (soon!)
